Decrypt

In order to decrypt an encrypted message, you'll need:

  • Your Private Key
  • The Public Key of the recipient (optional, for verification).
  • The encrypted text file file

Once you fill up the required fields, click on "Decrypt".

After decryption, you should be able to read the text message and you'll see an option to download the files.
